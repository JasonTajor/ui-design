# <Page Type> Template

> Copy for each page type. A template is a *recommended recipe*, not a rigid law —
> it tells Claude the default structure so pages of the same type stay consistent.

## When to use
What kind of page this is for.

## Sections (top → bottom)
The standard sections in order.

## Components
Which library components compose each section (reuse, don't invent).

## Spacing
Section separation and inner spacing, in tokens (e.g. sections `space-64` apart).

## Responsive rules
Breakpoint behavior: what stacks, what hides, what reflows.

## Animation rules
What animates on load/interaction, using motion tokens. Respect reduced-motion.
